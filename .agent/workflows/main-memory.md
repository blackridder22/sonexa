description: Everything you should know about the whole project. 
---

	•	The project name is Sonexa.
	•	Sonexa is a desktop app for video editors to manage both music and sound effects.
	•	The app must support offline mode and cloud sync using Supabase.
	•	The app is built with Electron + React + Tailwind + SQLite + Supabase.
	•	Core features: search, preview, tagging, drag & drop to Premiere or Resolve.
	•	The code must remain simple, clean, and easy to debug.
	•	Avoid unnecessary abstractions or complex patterns.
	•	The settings page is opened with CMD+, and handles API key, local path, and cache.
	•	All development and examples should focus only on building Sonexa.
	•	Keep all outputs short, direct, and focused on building the desktop app.
